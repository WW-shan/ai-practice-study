# 执行计划：AI 实践逐考点学习站

仓库名：`ai-practice-study`

## 第一轮：建站与全考点覆盖

1. 整理复习要点，拆成独立考点。
2. 每个考点回到对应课件或复习资料提取证据。
3. 按固定结构生成学习内容：课件证据、课件意图、从零讲解、考试考法、易错点、标准答案、小自测。
4. 对该考点自审，通过后进入下一个考点。
5. 创建 GitHub Pages 学习站并部署。

## 第二轮：严格精修与题目归位

1. 重新扫描 PPT 提取文本、复习资料、样例题和补充题。
2. 给每个考点补充“PPT 严格补充”模块。
3. 把浙大样例题、网络样例题按知识点放到正确页面。
4. 把命题/谓词符号化资料放入量词与谓词考点。
5. 把谓词推理 4 个证明题放入谓词推理考点。
6. 把 Alpha-Beta 剪枝图片题放入剪枝计算考点。
7. 逐考点二轮 review，确认课件、补充材料、题目和自测齐全。
8. 推送 GitHub Pages 并验证线上站点。
